XML Rules

XML has rules that must be followed in order to be a valid xml file. Describe 6 of these with code examples

XML Must Have Root Element

Each XML document has to have a root element. This element (Tag) contains the XML content.(“Nine Rules for Creating Good XML > Rule 1: All XML Must Have a Root Element,” n.d.)

<root> <author>John Soap</author> <author>Travis Kid</author> <author>Terry Fisher</author> </root>

Tags Must Be Closed

You have to close a tag that has been opened but if you do not close the tag it will break the code.(“Rule 2: All Tags Must Be Closed > Nine Rules for Creating Good XML,” n.d.)

Correct:

<author>Travis Falkenberg</author>

Wrong:

<author>Travis Falkeneberg<author>

Tags Must be Correctly Nested

When you open a tag(Child tag) inside another tag(Parent tag) you must make sure you close the child tag before you close the Parent tag.(“Rule 3: All Tags Must Be Properly Nested > Nine Rules for Creating Good XML,” n.d.)

Right:

<A><B>Test</B></A> <b><c>Test</c></b>

Wrong:

<A><B>Test</A></B> <b><c>Test</b></c>

Tags are Case Sensitive

Tags of the same name but one starts with a capital letter will be different from the tag that starts with a lower case. Opening tags and closing tags must match exactly.(“Rule 5: Tag Names Are Case Sensitive > Nine Rules for Creating Good XML,” n.d.)

Wrong:

<author>Test</Author> <AUTHOR>Test</aUTHOR>

Right:

<Author>Test</Author> <AUTHOR>Test</AUTHOR>

Tag Names Have Strict Linits

There are quite a few rules and restrictions that the naming convention needs to follow. The Letters XML can not be used at the beginning of the name of a tag. Numbers and punctuation are also not allowed in the beginning of a tag name.(“Rule 4: Tag Names Have Strict Limits > Nine Rules for Creating Good XML,” n.d.)

Wrong:

<01_Author>Test</01_Author> <"Author">Test</"Author">

Right:

<author>Test</author> <_author>Test</_author>

Tag Names Cannot Contain Spaces

Spaces in tag names can cause many problems with the data therefore they are prohibited in XML.(“Rule 6: Tag Names Cannot Contain Spaces > Nine Rules for Creating Good XML,” n.d.)

Right:

<author>Test</author>

Wrong:

<a uthor>Test</a uthor>
